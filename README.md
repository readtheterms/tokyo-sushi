# Tokyo Sushi — Website

A website for Tokyo Sushi (Redlands, CA) showing the menu, hours,
phone number, and address.

## Files

| File            | What it is                                                        |
|-----------------|-------------------------------------------------------------------|
| `index.html`    | The web page itself                                               |
| `menu-data.js`  | menu items, prices, hours, phone, address.                        |
| `styles.css`    | Colors and layout.                                                |
| `app.js`        | Code that builds the page from `menu-data.js`. Don't edit.        |

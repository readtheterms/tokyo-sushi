# Tokyo Sushi — Website

A simple, fast website for Tokyo Sushi (Redlands, CA) showing the menu, hours,
phone number, and address.

## Files

| File            | What it is                                                        |
|-----------------|-------------------------------------------------------------------|
| `index.html`    | The web page itself.                                              |
| `menu-data.js`  | **The only file you edit** — menu items, prices, hours, phone, address. |
| `styles.css`    | Colors and layout. You rarely need to touch this.                |
| `app.js`        | Code that builds the page from `menu-data.js`. Don't edit.       |

## How to preview it on your computer

Double-click `index.html` — it opens in your web browser. That's it.

## How to update the menu, prices, hours, phone, or address

1. Open `menu-data.js` in any text editor (Notepad works).
2. Follow the instructions written at the top of that file.
3. Save the file.
4. Refresh the web page (or re-open `index.html`) to see the changes.

**Tip:** Make a copy of `menu-data.js` before big edits, so you can undo if needed.

## How to put it on the internet (so customers can find it)

The site is just plain files, so it can be hosted for free. Easiest options:

- **Netlify Drop** — go to https://app.netlify.com/drop and drag this whole
  folder onto the page. You'll get a live web address in seconds. (Free.)
- **GitHub Pages** — if you use GitHub.
- **Your own domain** — most web hosts let you upload these files via their
  control panel or FTP. Point your domain (e.g. `tokyosushiredlands.com`) at them.

Once hosted, you can also tie it to your **Google Business Profile** so it shows
up when people search for the restaurant.
